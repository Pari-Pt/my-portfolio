About this

Portfolio Website

Front-End Web Developer, based in Portugal
Purpose

I created this website to introduce myself as a Web Developer, showcase my programming skills, and provide a means of contacting me for business/collaboration opportunities.

Code
Construction

I created a wireframe using Invision, which I then translated into basic HTML, along with some CSS. As my website grew, my wireframe design needed to adapt to faciliate a better flow and look. I also took this opportunity to experiment with different styling. I built my Portfolio with HTML, CSS and Vanilla JavaScript. Bootstrap technology was also implemented, the main example being the Navigation Bar which I tailored to my needs. For my editor, I utilised Visual Studio Code, which was tracked by my GitHub.

My experiences from this project

This was an exciting project with a great deal of content and it presented challenges with regards to page consistency, ease-of-use and ensuring everything was Responsive to different viewport sizes e.g. mobile devices. This required my integration of media queries for max-width, as well as breakpoints in grids e.g. "col-md-6". I thoroughly enjoy coding in JavaScript, as is clear from my Weather App, but my Portfolio project was an opportunity to focus on developing my command of HTML and CSS instead. Later on, I decided that there was a small need for JavaScript: the Day and Night mode buttons on the page About the Weather App. In the near future I will be increasing the presence of JavaScript by adding a contact form.

Achievements during this project:
  • Identified and addressed bugs, breaks and problems independently
  • Resolved the above or found an alternate method independently
  • Used media queries to alter CSS for different viewport sizes
  • Used :root and CSS variables to streamline code e.g. color and font-family
  • Tailored Bootstrap code, such as Nav Bar and Utility Classes to suit my own needs
  • Enhanced my command of building Responsive pages, especially with content such as grids, images and videos
  • Regular commits on my GitHub Repository explaining my changes to code
  • Developed my efficiency working on a large-scale (entire website functionality and flow) and also smaller scale (behaviour of individual pages and elements)
