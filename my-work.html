My Work

A weather app that allows users to obtain live weather data (via API calls) for a desired location or for the location of the user via geolocation. It also features a 5-day forecast, and changes color depending on whether it is daytime or nighttime in the searched location. This app also features a couple of festive themes for Christmas, Easter and Halloween.
